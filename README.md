# 🧮 Python Calculator

## 📌 Description

This is a simple command-line calculator built using Python.
It allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and more.

---

## 🚀 Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulo
* Exponentiation
* Floor Division
* Continuous menu using loop

---

## 🛠️ Technologies Used

* Python

---

## 📥 Clone Repository

To download this project, run:

```bash
git clone https://github.com/fatima-rifa/python-calculator.git
```

---

## ▶️ How to Run

1. Install Python
2. Download or clone this repository
3. Open terminal and run:

```bash
python calculator.py
```
